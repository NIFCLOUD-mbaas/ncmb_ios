Inherits from NCMBObject : NSObject
Declared in NCMBUser.h

Overview

NCMBUserクラスは、ニフクラ mobile backend上に保存されたユーザデータを管理するクラスです。 ユーザの新規登録やログイン/ログアウト、会員情報の更新・取得・削除を行います。 また、パスワードリセットやメールアドレス認証のリクエスト送信も行います。
